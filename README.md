# 1.Stockcoin is a bitcoin-like system
# 2.You can mine with CPU
    step:
    1.run stockcoin-qt
    2.click Help
    3.click Debug window
    4.click Console
    5.Type the command: setgenerate true

    Then,your computer starts mining

    挖矿步骤:
    1.运行 stockcoin-qt
    2.点击『帮助』
    3.点击『调试窗口』
    4.点击『控制台』
    5.在控制台下方，输入挖矿命令: setgenerate true

    至此，您的电脑已经自动开始挖矿
